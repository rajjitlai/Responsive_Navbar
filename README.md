# Responsive Navbar Design with Dynamic Animation

A sleek, modern, and fully responsive navigation bar featuring a unique "Smoking Animation" background. Built with semantic HTML5, CSS3, and vanilla JavaScript, this project demonstrates a premium user interface with smooth transitions and glassmorphism effects.

## ✨ Features

- **Dynamic Background**: A unique "Smoking Animation" achieved through an animated multi-color linear gradient.
- **Glassmorphism UI**: Uses `backdrop-filter` for a modern, frosted-glass effect in the header and mobile menu.
- **Fully Responsive**: Seamlessly adapts to desktop, tablet, and mobile screens.
- **Smooth Sidebar**: A slide-out navigation menu for mobile devices with interactive open/close animations.
- **Icon Integration**: Leveraging Unicons for a clean and professional visual aesthetic.
- **Zero Dependencies**: Pure HTML, CSS, and JS (no bulky frameworks).
